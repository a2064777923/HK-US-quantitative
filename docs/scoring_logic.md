# 信號引擎 v4 評分邏輯詳解

## 總覽

最終分 = (趨勢×30% + 動量×25% + 結構×20% + 量能×15% + 位置×10% + 1) ÷ 2

每個因子範圍 -1 ~ +1，最終分映射到 0 ~ 1。

## 1. 趨勢因子（30%）

### MA排列評分
| 條件 | 分數 | 含義 |
|------|------|------|
| 價 > MA5 > MA10 > MA20 | +1.0 | 強勢多頭排列 |
| 價 > MA5 > MA10 | +0.6 | 短中期多頭 |
| 價 > MA5 | +0.3 | 站上5日線 |
| 價 < MA5 < MA10 < MA20 | -1.0 | 強空頭排列 |
| 價 < MA5 < MA10 | -0.6 | 短中期空頭 |
| 價 < MA5 | -0.3 | 跌破5日線 |

### MA20斜率修正
- MA20 5日上升 > 2% → 趨勢分 +0.3
- MA20 5日下降 > 2% → 趨勢分 -0.3

## 2. 動量因子（25%）

### RSI(14) — Wilder's Smoothing
| RSI區間 | 分數 | 含義 |
|---------|------|------|
| < 25 | +0.4 | 超賣反彈機會 |
| 25-35 | +0.2 | 偏低 |
| 65-75 | -0.2 | 偏高 |
| > 75 | -0.4 | 超買回調風險 |

若 14 期平均漲幅和平均跌幅同時為 0，RSI 按中性 50 處理；只有平均跌幅為 0 且存在平均漲幅時才按 100 處理。

### MACD(12,26,9)
| 狀態 | 分數 | 含義 |
|------|------|------|
| MACD線>0 且 柱>0 | +0.3 | 金叉+正值 |
| 柱>0 但 MACD線<0 | +0.15 | 底部回升 |
| MACD線<0 且 柱<0 | -0.3 | 死叉+負值 |
| 柱<0 但 MACD線>0 | -0.15 | 頂部回落 |
| 前一柱≤0 且當前柱>0 | +0.2 | 剛金叉 |
| 前一柱≥0 且當前柱<0 | -0.2 | 剛死叉 |

### 5日動量
| 動量 | 分數 |
|------|------|
| > 5% | +0.2 |
| 2-5% | +0.1 |
| -5%~-2% | -0.1 |
| < -5% | -0.2 |

## 3. 結構因子（20%）

### 布林帶(20日, 2σ)
| 位置 | 分數 |
|------|------|
| 觸及下軌 | +0.3 (支撐) |
| 下半部 (BB% < 30%) | +0.15 |
| 觸及上軌 | -0.2 (壓力) |

## 4. 量能因子（15%）

### 量比 (當日量 / 20日均量)
| 條件 | 分數 |
|------|------|
| 價漲 + 量比>1.5 | +0.4 (放量上漲) |
| 價漲 + 量比>1.2 | +0.2 (量增價升) |
| 價跌 + 量比>2.0 | -0.3 (放量下跌→出貨) |
| 價跌 + 量比>1.5 | -0.15 (量增價跌) |

## 5. 位置因子（10%）

### 60日價格分位
| 分位 | 分數 |
|------|------|
| < 20% | +0.3 (低位) |
| > 85% | -0.2 (高位) |

## 硬規則（安全網）

1. **強空頭禁BUY**: 趨勢分 ≤ -0.6 → 最高分壓到 0.45
2. **MACD+RSI雙確認**: MACD<0 且 RSI<40 → 最高分壓到 0.48

## 掛單價計算

### 買入信號
- **掛單價**: min(第一支撐位, 現價×0.97)
- **止損價**: max(第二支撐位, 入場價×0.92)
- **止盈價**: max(第一阻力位, 入場價 + 3×ATR)

### 賣出信號
- **掛單價**: max(第一阻力位, 現價×1.03)
- **止損價**: min(第二阻力位, 入場價×1.08)
- **止盈價**: min(第一支撐位, 入場價 - 3×ATR)

### 支撐阻力來源
1. 近20日高低點
2. Classic Pivot Points: S1=2P-H, R1=2P-L, S2=P-(H-L), R2=P+(H-L)
3. MA5/10/20 作為動態支撐阻力
4. 布林帶上下軌

## 小時級預測

- 小時波動率 = 日均波動率 ÷ 6.5
- 趨勢drift = 趨勢方向 × 小時波動 × 時段數 × 0.3（保守）
- 不確定性 = 小時波動 × 時段數 × 0.7
- 預測區間 = 現價 × (1 ± drift ± uncertainty)
